# Referencias visuales y de estilo

Esta carpeta contiene material usado únicamente como referencia privada para el diseño del curso.

## Uso permitido dentro del proyecto

Estas referencias pueden usarse para inspirar:

- ritmo de explicación
- composición visual
- tipo de transiciones
- densidad de texto
- estructura narrativa
- uso moderado de humor técnico
- forma de explicar conceptos complejos de manera breve

## Restricciones

No copiar directamente contenido de terceros en materiales públicos del curso.

No redistribuir videos, capturas, imágenes, transcripciones o assets de terceros como parte del material final.

No usar capturas de pantalla, imágenes, frases o fragmentos de video en slides públicas salvo que exista permiso, licencia compatible o justificación clara de uso permitido.

## Regla para IA / Codex

Cuando una herramienta de IA lea esta carpeta, debe tratarla como referencia de estilo, no como fuente reutilizable.

La IA puede inferir patrones generales, por ejemplo:

- slides con poco texto
- ritmo visual rápido
- explicaciones concisas
- analogías simples
- humor técnico moderado

La IA no debe copiar:

- frases exactas
- imágenes
- screenshots
- layouts idénticos
- transcripciones
- secuencias completas
- marcas visuales distintivas de terceros

## Objetivo

Crear materiales originales para la materia **Sistemas Avanzados de Bases de Datos**, inspirados en buenas prácticas visuales y de comunicación técnica, pero sin replicar contenido protegido.
