# Template Assignment Repo

## Status

Template local reutilizable para repos `assignment-*`.

## Purpose

Servir como base comun para tareas evaluables fuera de clase.

## Convencion de Rama

Cada alumno debe trabajar en su propia rama:

```txt
student/<github-username>
```

## Quick Start

Levantar el entorno:

```bash
docker compose up -d
```

Entrar con `psql`:

```bash
psql postgresql://postgres:postgres@localhost:5432/demo
```

Alternativa visual opcional:

* DBeaver
