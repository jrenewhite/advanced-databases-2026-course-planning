# Templates de Issues

## Status

Templates canonicos locales.

## Purpose

Mantener en un solo lugar los issue templates reutilizables del curso.

## Uso

Copiar los templates necesarios desde:

```txt
.github/ISSUE_TEMPLATE/
```

hacia el repo concreto cuando aplique.

## Tipos Disponibles

* `evaluacion-practica.md`
* `actividad-guiada.md`
* `bug-simulado.md`
* `revision-tecnica.md`
* `proyecto-final.md`

## Regla Importante

No duplicar estos templates dentro de `assignment-repo/`.
